1.What is the  differnce btw sass and scss?
->Sass is the syntacally awesome style sheet used for the well organised css! Sass actually makes CSS more powerful with math and variable support. 
->SASS follows strict indentation
->SASS has a loose syntax with white space and no semicolons
->SASS file extension is .sass

->Scss code syntax is not required for SCSS.
->SCSS has no strict indentation.
-> the SCSS resembles more to CSS style and use of semicolons and braces are mandatory.
-> SCSS file extension is .scss.
